# Repositorio-de-Programaci-n-


Este proyecto es una pequeña aplicación hecha en Python que te permite ver, en tiempo real, los valores de TDS que envía un Arduino.  
La idea es ayudarte a saber si el agua está dentro de un nivel aceptable o si supera el umbral de contaminación.

La interfaz muestra:
- El valor actual del TDS.
- Una gráfica que se actualiza sola.
- Una alerta visual cuando el nivel es alto.
- Botones para conectar el Arduino, guardar datos y reiniciar el gráfico.
